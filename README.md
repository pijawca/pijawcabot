## Установка и запуск

```python
python3 -m venv venv
source venv/bin/activate
pip install get-pip.py
pip install -r requirements.txt
python3 bot.py
```

## **Настройка**

```
. .\config.py
    TOKEN = '' # @BotFather
    DB_NAME = '' # Название вашей базы
    DB_USER = '' # Название пользователя для базы
    DB_PASSWORD = '' # Пароль для вашего пользователя
    DB_HOST = 'localhost'
    DB_PORT = '5432'
```
